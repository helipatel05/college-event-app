🎓 College Event Management App:

A full-stack web application designed to manage college event registrations efficiently. This project uses Python (Flask) for the backend, MySQL for data storage, and a modern Frontend (HTML/CSS/JS).

🚀 Features:

User Registration: Students can register for specific events.
Real-time Data Storage: All registration details are stored in a MySQL database.

Responsive UI: Works perfectly on both Desktop and Mobile.

🛠️ Tech Stack:

Frontend: HTML5, CSS3, JavaScript
Backend: Python (Flask/Django)
Database: MySQL (Local &  MySQL Workbench)
Version Control: Git & GitHub

📂 Project Structure:
college-event-app/

    * static               # CSS and I
    * templates            # HTML files 
    * app.py               # Main Python Backend
    * requirements.txt     # Python Dependencies
    * README.md            # Project Documentation
